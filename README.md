## 🧮 BMI Formula

**Metric:**
```
BMI = weight(kg) / (height(m)²)
```

**Imperial:**
```
BMI = (703 × weight(lbs)) / (height(in)²)
```

## 📊 WHO BMI Categories

| Category      | BMI Range  | Color |
|---------------|------------|-------|
| Underweight   | < 18.5     | 🔵 Light Blue |
| Normal        | 18.5–24.9  | 🟢 Green |
| Overweight    | 25–29.9    | 🟠 Orange |
| Obese         | ≥ 30       | 🔴 Red |
